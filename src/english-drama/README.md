# 看剧学英语

这个专栏用来记录我通过影视剧学习英语的过程。每一节只围绕一集或一个片段，不追求把所有台词都记下来，而是提炼真正值得复习的表达。

## 当前剧集

- [《Agatha Christie's Poirot》（大侦探波洛）](/tech/english-drama/poirot/)

## 学习方法

每一节按六个部分整理：

1. 剧集信息：英文标题、季集、播出时间和案件背景。
2. 剧情入口：用中文快速回忆这一集讲什么。
3. 核心词汇：挑选能在日常表达中复用的单词和短语。
4. 重点句式：拆出可以替换使用的表达结构。
5. 听力观察：记录英式发音、语速、语气和场景用语。
6. 复习任务：把词汇和句式变成可执行的小练习。
