# 很适合资料党的浏览器插件：Notedown 脑袋笔记

平时看到一篇好文章，你会怎么处理？

很多人会点一下收藏夹，想着“以后再看”。结果过几个月再打开，链接失效了，网页改版了，标题也忘了。收藏夹越堆越多，能找回来的内容却越来越少。

最近我试了一款浏览器插件，叫 **Notedown · 脑袋笔记**。它的思路很实用：看到值得保存的网页，直接把正文提取出来，变成可以离线阅读、全文搜索、同步到 Git 仓库的笔记。

官网地址：[https://notedown.mnt.ltd/](https://notedown.mnt.ltd/)

## 一、它解决了什么问题

普通书签保存的是链接，Notedown 保存的是内容。

你收藏一篇文章时，它会自动识别网页正文，过滤广告、侧边栏和多余元素，把主要内容保存下来。以后即使原网页打不开，你本地仍然有一份可读版本。

它也支持全文搜索。标题、正文、标签、网址都可以查，而且离线可用。对经常查资料的人来说，这一点很有价值。

比如这些场景都很适合：

做研究时保存论文解读、技术博客、政策资料；  
写文章时保存素材、案例和观点片段；  
做项目时保存 GitHub Issue、教程、文档页面；  
做法律、合规、咨询工作时保存公告、法规、新闻和网页证据线索。

## 二、它和普通收藏工具的区别

我觉得 Notedown 最有意思的地方，是它把数据放回用户手里。

它默认把内容保存在浏览器本地。你不配置同步，也能当一个离线资料库使用。

如果想跨设备同步，可以把它接到自己的 Git 仓库，比如 GitHub、Gitee 或者私有化部署的 Gitea。保存后的内容会以 Markdown 或 HTML 文件形式进入仓库。

这意味着几件事：

你的资料可以自己备份；  
每次修改可以通过 Git 留痕；  
Markdown 文件可以继续放进 Obsidian、VS Code、静态博客或团队 Wiki；  
团队也可以用同一个仓库沉淀公共资料。

对重视数据控制权的人来说，这比把所有笔记都锁进某个云服务里舒服很多。

## 三、安装方法

目前推荐使用开发者模式安装。

打开 Notedown 官网或 GitHub Release 页面，下载插件压缩包：

[https://github.com/mnt-ltd/notedown/releases](https://github.com/mnt-ltd/notedown/releases)

下载后解压，然后按下面步骤操作：

1. 打开 Chrome，访问 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择刚才解压出来的插件目录
5. 把 Notedown 图标固定到浏览器右上角

安装完成后，打开任意文章页面，点击插件图标，就可以开始保存。

## 四、两种常用保存方式

第一种是保存整页。

打开一篇文章，点击 Notedown 图标，插件会自动提取正文。进入预览页后，可以修改标题、删掉多余内容、添加标签，最后确认保存。

第二种是保存局部内容。

在网页上右键，选择 Notedown 的“点选并保存”。页面会进入选择模式，鼠标移动到哪里，插件就会高亮对应区域。点击目标段落后，它会把这部分内容单独保存下来。

这个功能很适合保存长网页里的某一段内容，比如知乎回答、GitHub Issue 评论、公告里的关键条款、文章里的某个案例。

## 五、建议这样配置 Git 同步

如果只是临时试用，可以先不配置 Git。

如果准备长期使用，建议单独建一个私有仓库，比如 `my-notes` 或 `notedown-archive`，然后在插件设置里填写：

`Repo Owner`：用户名或组织名  
`Repo Name`：仓库名称  
`Branch`：一般是 `main` 或 `master`  
`Namespace`：笔记存放目录  
`Token`：Git 平台生成的访问令牌

Token 只给目标仓库读写权限即可，不要给过大的权限。仓库也建议设为私有。

配置完成后，Notedown 保存的笔记就能同步到你的仓库里。后续想做静态知识库、全文搜索、团队资料站，都有继续扩展的空间。

## 六、AI 功能可以按需开启

Notedown 还支持 OpenAI 兼容接口，可以用来做摘要、翻译、润色。

这个功能适合处理英文技术文档、长篇报告或资料摘录。需要注意的是，启用 AI 后，相关内容会发送到你配置的 AI 服务商，所以敏感资料要谨慎使用。

## 七、我推荐给谁

如果你只是偶尔收藏网页，普通收藏夹也够用。

如果你经常保存资料、写文章、做研究、查政策、整理项目文档，Notedown 值得试一下。

它的优势不在花哨界面，而在三个点：保存正文、离线搜索、数据可控。

很多工具让人越用越依赖平台。Notedown 的路径相反，它把内容整理成开放文件，再交还给用户自己管理。

对资料党来说，这一点很难得。
