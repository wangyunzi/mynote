按现行《民法典》、2024 年 7 月 1 日施行的新《公司法》、2023 年修正《民事诉讼法》及相关司法解释口径进行修正。

适用目标：客观题拿分、主观题可写、民法-商法-民诉一体化复习。

核心原则：不要把民法、商法、民诉拆成三个孤岛。真实案例通常是“实体权利 + 商事外观 + 程序实现”的组合题。

---

## 第一部分 学习方法：主客一体、民商诉融合

### 一、底层逻辑

传统复习容易出现两个问题：

1. 主客割裂：前期只刷选择题，考完客观题后才开始练主观题，导致能选不能写。
2. 学科割裂：民法、商法、民诉分开学，遇到股权转让、公司担保、执行异议、破产撤销等综合题时不会串联。

新的复习方式：

1. 以主观题标准倒逼客观题输入。
2. 以法律关系为主线打通民法、商法、民诉。
3. 每个考点都按“请求权基础 - 构成要件 - 抗辩 - 证明责任 - 程序出口”整理。

### 二、三段论答题法

每个问题都用四步：

1. 初步结论：甲能否向乙主张某项权利。
2. 大前提：列出法律规范和构成要件。
3. 小前提：把题干事实涵摄进构成要件。
4. 最终结论：支持、部分支持或不支持，并说明责任方式。

主观题标准句：

> 甲可依据某法律关系向乙主张某请求权。该请求权成立须具备 A、B、C 要件。本案中，……故该请求权成立/不成立。乙可否主张某抗辩，取决于……本案中……故法院应……

### 三、三维笔记法

每个知识点做成三栏：

| 案例事实 | 法条/规则 | 推导 |
|---|---|---|
| 谁和谁发生什么纠纷 | 请求权基础、构成要件、例外 | 为什么成立、为什么不成立、谁举证、怎么判 |

不要只记“结论”，要记“结论怎么来”。

### 四、强化期训练

每做一道客观题，至少追问四个问题：

1. 如果改主体，结论是否变化？
2. 如果改时间，是否涉及诉讼时效、除斥期间、保证期间？
3. 如果改标的，物权变动或担保顺位是否变化？
4. 如果进入诉讼，谁是原告、被告、第三人，谁承担举证责任？

### 五、冲刺期训练

每周做一次民商诉混合案例。训练目标不是“背答案”，而是：

1. 快速识别法律关系。
2. 快速定位请求权基础。
3. 快速判断程序路径。
4. 快速写出三段论。

## 第二部分 民商诉总框架

### 一、民商诉案件分析总图

1. 先看主体：自然人、法人、非法人组织、公司股东、董监高、管理人、保险人、票据持票人。
2. 再看关系：合同、物权、担保、侵权、婚姻继承、公司治理、破产、票据、保险。
3. 再看请求：给付、确认、形成、损害赔偿、排除妨碍、停止执行、优先受偿。
4. 再看抗辩：无效、撤销、未届期、同时履行、先诉抗辩、诉讼时效、善意取得、商事外观。
5. 最后看程序：管辖、当事人列置、举证责任、一审二审再审、执行、破产、第三人撤销之诉、执行异议之诉。

### 二、请求权基础优先级

民商诉主观题先找请求权基础：

1. 合同请求权：履行、违约金、损害赔偿、解除、返还。
2. 物权请求权：返还原物、排除妨碍、消除危险、恢复原状。
3. 担保请求权：保证责任、抵押/质押/留置优先受偿、让与担保清算。
4. 侵权请求权：赔偿损失、停止侵害、消除影响、赔礼道歉。
5. 公司法请求权：出资责任、股东代表诉讼、决议撤销/无效/不成立、人格否认。
6. 破产法请求权：取回权、别除权、抵销权、破产撤销权、申报债权。
7. 程序性请求：确认权属、停止执行、撤销生效裁判、申请执行、申请再审。

## 第三部分 民法核心复习

### 一、总则

#### 1. 民法基本原则

| 原则 | 识别关键词 | 常考场景 |
|---|---|---|
| 平等 | 主体地位平等、平等保护 | 国家机关作为民事主体签约 |
| 自愿 | 意思自治、不得强迫 | 合同订立、婚姻自愿 |
| 公平 | 权利义务明显失衡 | 显失公平、格式条款 |
| 诚信 | 善意履行、禁止权利滥用 | 恶意行权、反悔、虚假交易 |
| 公序良俗 | 公共秩序、善良风俗 | 代孕、悖德赠与、违法约定 |
| 绿色 | 节约资源、生态保护 | 相邻关系、环境侵权 |

#### 2. 民事法律关系

判断是否属于民事法律关系：

1. 主体是否平等。
2. 内容是否为私法上的人身、财产权益。
3. 是否产生民事权利义务。

国家机关也可能成为民事主体；关键不是身份，而是行为性质。

#### 3. 民事法律行为效力

有效要件：

1. 行为人有相应民事行为能力。
2. 意思表示真实。
3. 不违反法律、行政法规效力性强制规定。
4. 不违背公序良俗。

效力类型：

| 类型 | 典型情形 | 后果 |
|---|---|---|
| 有效 | 符合有效要件 | 按约履行 |
| 无效 | 无民事行为能力人行为、虚假意思表示、恶意串通、违法悖俗 | 自始无效，返还、折价、赔偿 |
| 可撤销 | 重大误解、欺诈、胁迫、显失公平 | 撤销前有效，撤销后自始无效 |
| 效力待定 | 限制行为能力人超范围行为、无权代理 | 追认后有效 |
| 未生效 | 需批准、登记等手续未完成 | 可能成立但未生效 |

撤销权期间：

1. 重大误解：知道或应当知道撤销事由之日起 90 日。
2. 欺诈、显失公平：知道或应当知道撤销事由之日起 1 年。
3. 胁迫：胁迫行为终止之日起 1 年。
4. 最长期间：行为发生之日起 5 年。

#### 4. 代理

代理题三问：

1. 是否有代理权？
2. 若无代理权，被代理人是否追认？
3. 是否构成表见代理？

表见代理成立：

1. 行为人无代理权。
2. 存在足以使相对人相信其有代理权的外观。
3. 相对人善意且无过失。
4. 行为后果归属于被代理人。

常考：离职人员、空白合同、公章管理、长期交易习惯。

#### 5. 诉讼时效

普通诉讼时效：3 年。最长保护期间：20 年。

不适用诉讼时效：

1. 停止侵害、排除妨碍、消除危险。
2. 不动产物权和登记动产物权权利人请求返还财产。
3. 请求支付抚养费、赡养费、扶养费。
4. 依法不适用的其他请求权。

区分：

| 制度 | 效果 |
|---|---|
| 诉讼时效届满 | 胜诉权抗辩，法院不主动适用 |
| 除斥期间届满 | 实体权利消灭，法院可主动审查 |
| 保证期间届满 | 保证责任消灭 |

### 二、物权

#### 1. 物权变动

| 标的 | 原则 | 例外 |
|---|---|---|
| 不动产 | 登记生效 | 继承、法院文书、征收等非基于法律行为取得 |
| 动产 | 交付生效 | 特殊动产未经登记不得对抗善意第三人 |

物权变动题要区分：

1. 合同效力：买卖合同是否有效。
2. 物权效力：所有权是否转移。
3. 对抗效力：能否对抗第三人。

#### 2. 善意取得

构成要件：

1. 处分人无处分权。
2. 受让人受让时善意。
3. 以合理价格转让。
4. 已经登记或交付。

常考陷阱：

1. 无权处分不当然导致合同无效。
2. 善意取得解决的是物权归属，不直接解决违约责任。
3. 赃物、遗失物、埋藏物适用特殊规则。

#### 3. 共有

按份共有：按份额享有权利。处分共有不动产或动产，一般需占份额三分之二以上按份共有人同意，另有约定除外。

共同共有：共同享有，不分份额。处分通常需全体共同共有人同意，另有约定除外。

#### 4. 居住权

要点：

1. 为满足生活居住需要。
2. 原则上无偿设立，另有约定除外。
3. 书面合同 + 登记设立。
4. 不得转让、继承。
5. 设立居住权的住宅原则上不得出租，另有约定除外。

### 三、担保制度

担保是民商诉融合高频区。

#### 1. 保证

保证方式：

1. 一般保证。
2. 连带责任保证。

民法典口径：约定不明时，推定为一般保证。

一般保证人享有先诉抗辩权；连带责任保证人不享有。

保证期间：

1. 有约定按约定。
2. 未约定或约定不明：主债务履行期限届满之日起 6 个月。

主观题写法：

> 保证合同有效。因当事人未明确约定连带责任保证，依法推定为一般保证，保证人享有先诉抗辩权。债权人仅起诉一般保证人的，法院应结合程序规则追加债务人参加诉讼。

#### 2. 抵押

抵押不转移占有。

| 财产 | 抵押权设立 |
|---|---|
| 不动产 | 登记设立 |
| 动产 | 抵押合同生效时设立，未经登记不得对抗善意第三人 |

抵押财产转让：

1. 抵押期间，抵押人可以转让抵押财产。
2. 抵押权不受影响，具有追及效力。
3. 抵押权人能证明转让可能损害抵押权的，可请求提前清偿或提存。

#### 3. 质押

质押一般转移占有或交付权利凭证。

动产质权自交付时设立。权利质权按对应权利类型交付或登记设立。

#### 4. 留置

留置权是法定担保物权。

成立：

1. 债权人合法占有债务人动产。
2. 债权已届清偿期。
3. 动产与债权属于同一法律关系，企业之间留置除外。

同一动产上抵押、质押、留置并存，留置权优先。

#### 5. 让与担保

典型：债务人把房产、股权形式上过户给债权人作为担保。

处理：

1. 不承认债权人直接取得所有权。
2. 承认担保功能。
3. 债权人正确诉请应是请求折价、拍卖、变卖所得价款优先受偿。
4. 直接诉请确认财产归自己所有，法院不支持。

### 四、合同

#### 1. 合同分析顺序

1. 是否成立。
2. 是否生效。
3. 内容如何解释。
4. 是否履行。
5. 是否存在抗辩权。
6. 是否解除。
7. 是否承担违约责任。
8. 是否与物权、担保、公司、程序发生交叉。

#### 2. 合同订立

要约：内容具体确定，并表明经承诺即受约束。

要约邀请：希望他人向自己发出要约。拍卖公告、招标公告、招股说明书、商业广告原则上为要约邀请；商业广告内容符合要约条件的，可构成要约。

承诺：到达要约人时生效。对要约作实质性变更，构成新要约。

#### 3. 格式条款

提供格式条款一方应合理提示免除或减轻其责任等重大利害关系条款。

无效情形：

1. 不合理免除或减轻己方责任。
2. 加重对方责任。
3. 限制或排除对方主要权利。

解释规则：

1. 通常理解解释。
2. 有两种以上解释的，作不利于提供格式条款一方解释。
3. 格式条款与非格式条款不一致的，采用非格式条款。

#### 4. 双务合同履行抗辩权

| 抗辩 | 适用 |
|---|---|
| 同时履行抗辩权 | 双方互负到期债务，无先后履行顺序 |
| 先履行抗辩权 | 有先后履行顺序，先履行方未履行或履行不符合约定 |
| 不安抗辩权 | 先履行方有确切证据证明后履行方履约能力严重恶化 |

#### 5. 合同保全

代位权：

1. 债权人对债务人享有到期债权。
2. 债务人怠于行使其到期债权或从权利。
3. 影响债权人债权实现。
4. 债权人以自己名义向法院行使。

撤销权：

1. 债务人无偿处分财产权益，影响债权实现。
2. 或以明显不合理低价转让、高价受让，且相对人知道或应当知道。
3. 知道或应当知道撤销事由之日起 1 年内行使；行为发生之日起 5 年内未行使消灭。

#### 6. 合同解除

法定解除：

1. 不可抗力致使合同目的不能实现。
2. 履行期届满前明确表示或以行为表明不履行主要债务。
3. 迟延履行主要债务，经催告后仍不履行。
4. 迟延履行或其他违约致使合同目的不能实现。
5. 法律规定其他情形。

#### 7. 违约责任

承担方式：

1. 继续履行。
2. 采取补救措施。
3. 赔偿损失。
4. 支付违约金。
5. 定金罚则。

损害赔偿范围：履行利益 + 可预见规则 + 减损规则。

定金不得超过主合同标的额 20%，超过部分不产生定金效力。

### 五、典型合同

#### 1. 买卖合同

重点：

1. 风险一般自交付时转移。
2. 所有权是否转移，要看动产交付、不动产登记。
3. 一物数卖要区分普通动产、特殊动产、不动产。
4. 出卖人瑕疵担保责任与违约责任可结合考。

一物数卖简化规则：

1. 普通动产：先受领交付者优先。
2. 特殊动产：先受领交付、先登记等规则结合判断。
3. 不动产：登记取得物权；未登记可能仅有债权请求权。

#### 2. 租赁合同

重点：

1. 租期不得超过 20 年。
2. 买卖不破租赁。
3. 承租人经出租人同意可以转租。
4. 房屋承租人优先购买权有例外。

#### 3. 借款合同

自然人之间借款合同，自贷款人提供借款时成立。

民间借贷重点看：

1. 借款是否实际交付。
2. 利息是否约定。
3. 利率是否超过司法保护上限。
4. 砍头息、复利、保证、夫妻债务是否交叉。

#### 4. 建设工程合同

高频点：

1. 无效施工合同，工程质量合格的，可参照合同约定折价补偿。
2. 承包人享有建设工程价款优先受偿权。
3. 优先受偿范围一般限于工程价款，不包括违约金和发包人违约造成的损失。
4. 行使期限：自发包人应当给付建设工程价款之日起，合理期限内行使，最长不超过 18 个月。

程序融合：

1. 发包人起诉质量修复费用，承包人可反诉工程款。
2. 法院应查明优先受偿的工程价款数额及相应工程，并在判决主文中明确。

#### 5. 保理合同

核心：

1. 债权人将现有或将有的应收账款转让给保理人。
2. 保理人提供资金融通、应收账款管理、催收等服务。
3. 债务人已向保理人确认应收账款真实性后，通常不得再以应收账款不存在对抗保理人，但仍要结合保理人善意和审查义务判断。

#### 6. 融资租赁合同

承租人违约时，出租人可请求支付全部租金，也可解除合同并取回租赁物。

取回租赁物价值超过欠付租金和其他费用的，超过部分应返还承租人。

### 六、人格权

人格权请求权重点：

1. 停止侵害。
2. 排除妨碍。
3. 消除危险。
4. 消除影响、恢复名誉。
5. 赔礼道歉。
6. 赔偿损失。

高频：

1. 肖像权：未经同意制作、使用、公开肖像，除合理使用外构成侵权。
2. 名誉权：侮辱、诽谤、虚假事实陈述。
3. 隐私权：私人生活安宁、私密空间、私密活动、私密信息。
4. 个人信息：合法、正当、必要、明示、同意、不得过度处理。

### 七、婚姻家庭与继承

#### 1. 夫妻共同财产

共同财产：

1. 工资、奖金、劳务报酬。
2. 生产、经营、投资收益。
3. 知识产权收益。
4. 继承或受赠财产，明确只归一方的除外。

个人财产：

1. 婚前财产。
2. 人身损害赔偿或补偿。
3. 明确只归一方的遗嘱或赠与财产。
4. 一方专用生活用品。

夫妻共同债务：

1. 共同签名。
2. 一方事后追认。
3. 一方为家庭日常生活需要所负债务。
4. 超出日常生活需要的，债权人需证明用于共同生活、共同生产经营或基于共同意思表示。

#### 2. 继承

继承顺序：

1. 第一顺序：配偶、子女、父母。
2. 第二顺序：兄弟姐妹、祖父母、外祖父母。

遗产处理优先级：

1. 遗赠扶养协议。
2. 遗嘱继承、遗赠。
3. 法定继承。

遗嘱形式：

1. 自书。
2. 代书。
3. 打印。
4. 录音录像。
5. 口头。
6. 公证。

民法典取消公证遗嘱优先效力；数份遗嘱冲突，以最后遗嘱为准。

---

### 八、侵权责任

一般侵权构成：

1. 行为。
2. 损害。
3. 因果关系。
4. 过错。

归责原则：

| 类型 | 场景 |
|---|---|
| 过错责任 | 一般侵权 |
| 过错推定 | 教育机构、物件损害等部分情形 |
| 无过错责任 | 产品责任、环境污染、高度危险、饲养动物等 |

高频特殊侵权：

1. 用人单位责任：工作人员执行工作任务造成损害，由用人单位承担；故意或重大过失可追偿。
2. 产品责任：被侵权人可向生产者或销售者请求赔偿。
3. 机动车事故：先交强险，再商业险或责任人。
4. 医疗损害：医疗机构过错责任，特定情形推定过错。
5. 环境污染：无过错责任，污染者对免责、减责和不存在因果关系举证。
6. 高空抛物坠物：公安等机关应调查；难以确定侵权人的，可能加害的建筑物使用人补偿。

---

## 第四部分 商法核心复习

### 一、公司法

#### 1. 公司人格独立与股东有限责任

公司以全部财产对公司债务承担责任。股东以认缴出资额或认购股份为限承担责任。

人格否认成立：

1. 股东滥用公司法人独立地位和有限责任。
2. 逃避债务。
3. 严重损害债权人利益。
4. 股东对公司债务承担连带责任。

典型：财产混同、业务混同、人员混同、过度控制。

一人公司特别规则：股东不能证明公司财产独立于自己财产的，应对公司债务承担连带责任。

#### 2. 出资责任

新公司法重点：

1. 有限公司注册资本原则上自公司成立之日起 5 年内缴足。
2. 股东未按期足额缴纳出资，应向公司补足，并对公司损失承担赔偿责任。
3. 公司不能清偿到期债务时，特定条件下未届期出资可能加速到期。

出资加速到期与入库规则：

1. 股东提前缴纳出资，应归入公司财产。
2. 原告债权人不能直接把股东出资拿走。
3. 程序上通常要通过公司债权实现和执行公司对股东的到期债权。

#### 3. 公司对外担保

现行公司法第 15 条规定，公司为他人提供担保，按章程规定由董事会或股东会决议；为股东或实际控制人提供担保，应经股东会决议，相关股东回避表决。

担保效力分析：

1. 法定代表人越权提供担保，先看相对人是否善意。
2. 善意通常要求相对人对公司决议进行合理形式审查。
3. 相对人未审查决议，通常难认定善意。
4. 担保对公司不发生效力时，转入担保无效后的过错赔偿。

程序攻防：

1. 债权人主张公司承担担保责任，应举证其已审查决议。
2. 公司抗辩担保无效，应说明担保未经合法决议且相对人非善意。
3. 法院需区分担保责任与担保无效赔偿责任。

#### 4. 公司决议效力

| 类型 | 典型情形 |
|---|---|
| 不成立 | 未召开会议、未表决、表决人数不够、无有效决议结果 |
| 无效 | 决议内容违反法律、行政法规 |
| 可撤销 | 召集程序、表决方式违反法律、章程，或内容违反章程 |

撤销期间：一般自决议作出之日起 60 日内。

#### 5. 股权转让

有限公司股东之间可相互转让。

向股东以外的人转让，重点看：

1. 是否通知其他股东。
2. 其他股东是否主张同等条件下优先购买权。
3. 章程是否另有规定。
4. 是否涉及股权代持、执行查封、夫妻共同财产、公司登记。

#### 6. 股东代表诉讼

适用：

1. 董监高或他人侵害公司利益。
2. 公司怠于起诉。
3. 符合条件的股东为公司利益以自己名义起诉。

程序：

1. 原则上先书面请求监事会或董事会起诉。
2. 被拒绝、30 日内未起诉，或情况紧急，可直接起诉。
3. 原告是股东，被告是侵害人，公司列第三人。
4. 胜诉利益归公司。

#### 7. 董事、高管责任

义务：

1. 忠实义务。
2. 勤勉义务。

禁止：

1. 挪用公司资金。
2. 擅自与公司交易。
3. 篡夺公司商业机会。
4. 自营或为他人经营同类业务。
5. 违规接受佣金。
6. 泄露公司秘密。

董事辞任与无因解任：

1. 董事与公司关系可类比委托关系。
2. 公司可依法解任董事，但无正当理由提前解任造成损失的，董事可请求赔偿。
3. 董事辞任导致董事会成员低于法定人数的，新董事就任前原董事仍应履职。

---

### 二、合伙企业法

普通合伙人：对合伙企业债务承担无限连带责任。

有限合伙人：以认缴出资额为限承担责任，不得执行合伙事务，不得对外代表有限合伙。

有限合伙人下列行为不视为执行事务：

1. 参与决定普通合伙人入伙、退伙。
2. 对经营管理提出建议。
3. 参与选择会计师事务所。
4. 获取财务报告。
5. 涉及自身利益时查阅账簿。
6. 权益受侵害时主张权利。

---

### 三、企业破产法

#### 1. 破产原因

企业法人不能清偿到期债务，并且资产不足以清偿全部债务或明显缺乏清偿能力。

#### 2. 受理效果

法院受理破产申请后：

1. 债务人财产由管理人接管。
2. 个别清偿原则上无效或可撤销。
3. 保全措施解除，执行程序中止。
4. 未履行完毕合同由管理人决定继续履行或解除。
5. 债权人应申报债权。

#### 3. 破产撤销权

可撤销行为：

1. 无偿转让财产。
2. 明显不合理价格交易。
3. 对无财产担保债务提供担保。
4. 对未到期债务提前清偿。
5. 放弃债权。
6. 特定期间内个别清偿。

程序重点：

1. 破产撤销权由管理人以自己名义行使。
2. 个别债权人不能直接行使破产撤销权。
3. 目的不是保护某个债权人，而是恢复债务人财产，公平清偿全体债权人。

#### 4. 取回权、别除权、抵销权

取回权：权利人取回不属于债务人的财产。

别除权：有担保债权人就担保财产优先受偿。

抵销权：债权人在破产申请受理前对债务人负有债务的，可依法抵销，但恶意取得债权、债务的受限制。

#### 5. 清偿顺序

先清偿破产费用和共益债务。

之后：

1. 职工工资、医疗伤残补助、抚恤费用，基本养老保险、基本医疗保险费用，补偿金。
2. 欠缴的除前项以外的社会保险费用和税款。
3. 普通破产债权。

---

### 四、票据法

票据特征：

1. 文义性。
2. 要式性。
3. 无因性。
4. 独立性。
5. 流通性。

票据无因性不是绝对：

1. 直接前后手之间，可以基于基础关系抗辩。
2. 持票人明知存在抗辩事由而取得票据的，债务人可抗辩。
3. 物的抗辩可对任何持票人主张。

程序融合：

票据追索权纠纷中，被告提出“基础合同未履行”抗辩时，法院可能需要审理基础合同关系。

---

### 五、保险法

#### 1. 如实告知义务

投保人故意或因重大过失未履行如实告知义务，足以影响保险人决定是否承保或提高保险费率的，保险人有权解除合同。

#### 2. 保险利益

财产保险：对保险标的有保险利益。

人身保险：投保人对被保险人有保险利益。以死亡为给付条件的人身保险，通常需被保险人同意并认可保险金额。

#### 3. 保险代位求偿

财产保险中，保险人赔偿后，在赔偿金额范围内代位行使被保险人对第三人的请求权。

程序：

1. 保险人以自己名义起诉第三者。
2. 保险人须证明已实际支付保险金。
3. 第三者可向保险人主张其对被保险人的抗辩。

---

## 第五部分 民事诉讼法核心复习

### 一、主管与管辖

民事诉讼主管：平等主体之间的人身关系、财产关系纠纷。

一般地域管辖：原告就被告。

特殊管辖：

1. 合同纠纷：被告住所地或合同履行地。
2. 侵权纠纷：侵权行为地或被告住所地。
3. 公司纠纷：公司住所地。
4. 票据纠纷：票据支付地或被告住所地。

专属管辖：

1. 不动产纠纷：不动产所在地。
2. 港口作业纠纷：港口所在地。
3. 继承遗产纠纷：被继承人死亡时住所地或主要遗产所在地。

协议管辖：

1. 合同或其他财产权益纠纷。
2. 书面协议。
3. 选择与争议有实际联系地点法院。
4. 不违反级别管辖和专属管辖。

### 二、当事人

常见列置：

| 案件 | 原告 | 被告 | 第三人 |
|---|---|---|---|
| 股东代表诉讼 | 股东 | 侵害公司利益者 | 公司 |
| 法人人格否认 | 债权人 | 公司 + 滥用权利股东 | 视案情 |
| 保险代位求偿 | 保险人 | 第三者 | 被保险人可视情况参加 |
| 案外人执行异议之诉 | 案外人 | 申请执行人；被执行人反对异议的也为被告 | 其他利害关系人 |
| 第三人撤销之诉 | 受损第三人 | 原审双方当事人 | 视案情 |

### 三、证据与证明责任

基本原则：谁主张，谁举证。

高频倒置或特殊分配：

1. 一人公司财产独立：股东自证公司财产独立。
2. 环境污染：污染者对免责、减责和不存在因果关系举证。
3. 公司担保：债权人主张善意，需证明已合理审查决议。
4. 医疗损害部分情形：医疗机构存在法定情形时推定过错。

### 四、一审、二审、再审

上诉期：

1. 判决：15 日。
2. 裁定：10 日。

再审重点：

1. 新证据足以推翻原判。
2. 基本事实缺乏证据。
3. 主要证据伪造。
4. 适用法律错误。
5. 违反法定程序。
6. 审判人员贪污受贿、徇私舞弊、枉法裁判。

### 五、执行与执行异议

执行救济：

1. 当事人、利害关系人执行行为异议。
2. 案外人对执行标的异议。
3. 案外人执行异议之诉。
4. 申请执行人执行异议之诉。

案外人执行异议之诉：

1. 核心是案外人对执行标的是否享有足以排除强制执行的民事权益。
2. 可一并请求确认权属，法院可合并审理。
3. 股权代持、未过户房屋买受人、让与担保、轮候查封是高频场景。

### 六、第三人撤销之诉

适用：

1. 第三人因不能归责于自己的事由未参加诉讼。
2. 生效判决、裁定、调解书部分或全部内容错误。
3. 损害其民事权益。
4. 自知道或应当知道权益受损之日起 6 个月内起诉。

高频：虚假诉讼、恶意调解、以房抵债、虚构债务。

## 第六部分 民商诉融合专题

### 专题 1 公司对外担保

实体：

1. 公司为他人担保，应按章程由董事会或股东会决议。
2. 为股东或实际控制人担保，应经股东会决议，相关股东回避。
3. 法定代表人越权担保，相对人善意的，担保对公司发生效力；非善意的，不发生担保效力，转入过错赔偿。

程序：

1. 债权人举证审查过决议。
2. 公司举证章程限制、未决议、相对人非善意。
3. 法院区分担保有效责任和担保无效赔偿责任。

答题句：

> 本案虽由法定代表人签订担保合同，但公司对外担保受公司法决议规则约束。债权人作为商事主体，应对公司决议进行合理形式审查。若其未审查，难认定为善意，担保对公司不发生效力；公司是否承担赔偿责任，应按双方过错确定。

### 专题 2 对赌协议

实体：

1. 投资方与股东对赌，一般按合同处理。
2. 投资方与目标公司对赌，合同效力与履行可能分离。
3. 公司回购股权涉及减资程序；金钱补偿可能涉及利润分配和资本维持。

程序：

1. 合同有效不等于可直接判令立即履行。
2. 法院可能作附条件给付，如完成减资程序后支付回购款。

### 专题 3 股权代持与隐名股东

内部：

1. 代持协议有效，隐名股东享有投资权益。

外部：

1. 商事外观主义保护信赖登记的第三人。
2. 隐名股东显名化通常需其他股东同意。
3. 名义股东债权人申请执行登记股权时，隐名股东能否排除执行，要看申请执行人是否基于登记外观发生交易、是否善意。

程序：

1. 隐名股东常提案外人执行异议之诉。
2. 不能排除执行的，只能向名义股东主张违约或赔偿。

### 专题 4 法人人格否认

实体：

1. 股东滥用法人独立地位和有限责任。
2. 逃避债务。
3. 严重损害债权人利益。
4. 股东对公司债务承担连带责任。

程序：

1. 通常公司和股东列共同被告。
2. 原则上债权人证明混同。
3. 一人公司由股东证明财产独立。

### 专题 5 建设工程价款优先受偿权

实体：

1. 承包人就建设工程折价或拍卖价款优先受偿。
2. 优先于抵押权和普通债权。
3. 优先范围一般限于工程价款，不包括违约金、损害赔偿等。
4. 最长 18 个月，自发包人应当给付建设工程价款之日起算。

程序：

1. 承包人可起诉、仲裁或以其他方式依法主张。
2. 法院应在判决主文中明确优先受偿的数额和对应工程。

### 专题 6 股东代表诉讼

实体：

1. 董监高或他人损害公司利益。
2. 股东为公司利益起诉。
3. 胜诉利益归公司。

程序：

1. 原则上履行书面请求前置程序。
2. 股东为原告，侵害人为被告，公司为第三人。
3. 情况紧急可不经前置程序。

### 专题 7 让与担保

实体：

1. 形式上转移财产权。
2. 实质上担保债务。
3. 禁止债权人直接取得所有权。

程序：

1. 债权人诉请确认财产归自己所有，不支持。
2. 正确诉请是请求折价、拍卖、变卖所得价款优先受偿。

### 专题 8 票据无因性与基础关系抗辩

实体：

1. 票据关系原则上独立于基础关系。
2. 直接前后手之间可基于基础关系抗辩。
3. 持票人恶意取得票据的，可被抗辩。

程序：

1. 案由虽为票据追索权纠纷，法院可能需要审查基础合同。
2. 持票人应证明票据权利，被告应证明抗辩事由。

### 专题 9 保险人代位求偿

实体：

1. 财产保险保险人赔偿后，代位取得被保险人对第三者的请求权。
2. 本质是法定债权转移。

程序：

1. 保险人以自己名义起诉第三者。
2. 保险人证明已赔付。
3. 第三者可主张对被保险人的抗辩。

### 专题 10 证券虚假陈述

实体：

1. 信息披露义务人虚假记载、误导性陈述或重大遗漏。
2. 投资者因证券交易遭受损失。
3. 适用特殊侵权责任。

程序：

1. 可适用代表人诉讼、特别代表人诉讼。
2. 特别代表人诉讼有“默示加入、明示退出”的制度设计。

### 专题 11 破产撤销权与民法撤销权

民法撤销权：

1. 债权人以自己名义行使。
2. 保护个别债权人债权实现。
3. 适用民法典合同保全规则。

破产撤销权：

1. 管理人以自己名义行使。
2. 保护全体债权人公平受偿。
3. 适用破产法特别规则。

### 专题 12 执转破

实体：

1. 企业法人不能清偿到期债务且资不抵债或明显缺乏清偿能力。
2. 个别执行应让位于破产公平清偿。

程序：

1. 执行法院发现符合破产条件，经申请执行人或被执行人同意，可移送破产审查。
2. 破产受理后，执行中止，保全解除，债权统一申报。

### 专题 13 执行异议之诉与确权

实体：

1. 案外人对执行标的主张所有权、物权期待权、担保权益等。
2. 关键看该权益是否足以排除强制执行。

程序：

1. 案外人可同时请求确认权属。
2. 法院可合并审理确权和排除执行。
3. 轮候查封场景中，通常以首封申请执行人或优先受偿权人为主要被告，其他利害关系人列第三人。

### 专题 14 阴阳合同与穿透式审判

实体：

1. 通谋虚伪意思表示下，表面行为无效。
2. 隐藏行为按其真实性质判断效力。

程序：

1. 法院发现当事人主张法律关系与查明事实不一致，应进行释明。
2. 当事人拒绝变更诉请的，法院仍应尽量根据真实法律关系裁判，避免程序空转。

### 专题 15 债权人撤销权的连环给付

实体：

1. 债务人不当处分财产影响债权实现。
2. 债权人可撤销处分行为，并请求相对人向债务人返还财产。

程序：

1. 可与债权债务关系合并审理。
2. 判决可能形成“相对人返还给债务人，债务人清偿债权人”的连环给付。

### 专题 16 代位权诉讼合并审理

实体：

1. 债务人怠于行使对相对人的到期债权，影响债权人实现。
2. 债权人以自己名义起诉相对人。

程序：

1. 多个债权人对同一次债务人提起代位权诉讼的，可合并审理。
2. 次债务人财产不足时，按债权比例确定履行份额。

## 第七部分 主观题模板

### 一、合同题模板

1. 甲乙之间合同是否成立并生效。
2. 乙是否构成违约。
3. 甲可否请求继续履行、解除、违约金、赔偿损失。
4. 乙有无抗辩，如合同无效、同时履行抗辩、不可抗力、诉讼时效。
5. 法院如何判。

标准句：

> 甲乙合同系双方真实意思表示，内容不违反法律、行政法规效力性强制规定及公序良俗，合法有效。乙未按约履行主要债务，构成违约。甲可依据民法典合同编请求乙承担继续履行、赔偿损失或支付违约金责任。乙主张……抗辩，因……不能成立/可以成立。故法院应……

### 二、物权题模板

1. 合同是否有效。
2. 是否完成登记或交付。
3. 是否发生物权变动。
4. 第三人是否善意取得或能否对抗。
5. 请求权是返还原物、排除妨碍、损害赔偿还是违约责任。

### 三、担保题模板

1. 主债权是否有效。
2. 担保合同是否有效。
3. 担保物权是否设立并能否对抗。
4. 是否存在保证期间、先诉抗辩、公司决议、善意审查问题。
5. 判决责任方式。

### 四、公司题模板

1. 先找公司法特别规则。
2. 公司法无规定，再回到民法典总则、合同、侵权。
3. 同时审查章程效力、决议效力、登记外观。
4. 程序上确定原告、被告、公司是否列第三人。

### 五、执行异议题模板

1. 执行标的是什么。
2. 案外人主张何种实体权利。
3. 该权利是否足以排除执行。
4. 是否涉及商事外观主义、登记公信、物权期待权。
5. 诉讼请求能否合并确权。

## 第八部分 复习计划

### 第一阶段：宏观搭建

目标：建立民商诉全景图。

任务：

1. 民法按“总则 - 物权 - 合同 - 人格 - 婚姻继承 - 侵权”过一遍。
2. 商法按“公司 - 破产 - 票据 - 保险”过一遍。
3. 民诉按“管辖 - 当事人 - 证据 - 审判 - 执行”过一遍。
4. 每学一个实体法点，同时标注程序出口。

### 第二阶段：微观夯实

目标：把核心考点做成请求权卡片。

每张卡片包含：

1. 考点名称。
2. 请求权基础。
3. 构成要件。
4. 易错点。
5. 程序点。
6. 主观题一句话。

优先做：

1. 民事法律行为效力。
2. 代理与表见代理。
3. 物权变动与善意取得。
4. 担保。
5. 合同履行、解除、违约。
6. 公司担保。
7. 股东出资。
8. 破产撤销。
9. 执行异议。

### 第三阶段：客观题检验

目标：用客观题查漏洞。

做题要求：

1. 不只看答案。
2. 每个选项都写一句理由。
3. 错题按“事实误读、规则记错、例外漏掉、程序混淆”分类。
4. 把错题改编成主观题小问。

### 第四阶段：主观题实战

目标：能写。

训练：

1. 每周至少 2 道民商诉融合案例。
2. 限时写。
3. 写完反查法条。
4. 对照答案补“请求权基础”和“程序出口”。

### 第五阶段：法条回归

重点看：

1. 民法典总则编、物权编担保、合同编通则、典型合同、侵权责任。
2. 公司法：出资、决议、担保、董监高、代表诉讼、解散清算。
3. 企业破产法：受理效果、撤销权、取回权、别除权、抵销权、清偿顺序。
4. 民诉法：管辖、当事人、证据、审判监督、执行异议、第三人撤销之诉。

重点记：

1. 但书。
2. 期间。
3. 举证责任。
4. 例外规则。
5. 判决主文怎么写。

## 第九部分 高频易混表

### 一、合同效力与物权变动

| 问题 | 判断 |
|---|---|
| 买卖合同未登记 | 合同通常有效，物权未变动 |
| 无权处分合同 | 不当然无效 |
| 抵押合同已生效但未登记 | 不动产抵押权未设立 |
| 动产抵押未登记 | 抵押权设立，但不得对抗善意第三人 |

### 二、一般保证与连带保证

| 项目 | 一般保证 | 连带保证 |
|---|---|---|
| 约定不明 | 推定一般保证 | 需明确约定 |
| 先诉抗辩权 | 有 | 无 |
| 起诉列置 | 债务人通常应参加 | 可直接起诉保证人 |

### 三、民法撤销权与破产撤销权

| 项目 | 民法撤销权 | 破产撤销权 |
|---|---|---|
| 行使主体 | 债权人 | 管理人 |
| 保护对象 | 个别债权人 | 全体债权人 |
| 法律依据 | 民法典合同保全 | 企业破产法 |
| 财产归属 | 回到债务人责任财产 | 回到破产财产 |

### 四、第三人撤销之诉与执行异议之诉

| 项目 | 第三人撤销之诉 | 执行异议之诉 |
|---|---|---|
| 针对对象 | 生效判决、裁定、调解书 | 执行行为或执行标的 |
| 目的 | 撤销错误裁判内容 | 排除或许可执行 |
| 期限 | 知道受损之日起 6 个月 | 执行异议裁定后法定期限内 |
| 高频场景 | 虚假诉讼 | 房屋、股权、代持、查封 |

### 五、公司人格否认与一人公司责任

| 项目 | 一般人格否认 | 一人公司财产混同 |
|---|---|---|
| 证明责任 | 债权人证明滥用和损害 | 股东证明财产独立 |
| 责任 | 股东连带责任 | 股东连带责任 |
| 高频证据 | 账户混用、业务混同、人员混同 | 审计报告、财务账簿 |

## 第十部分 最终背诵抓手

1. 总则看效力：民事法律行为、代理、时效。
2. 物权看公示：登记、交付、善意取得。
3. 合同看阶段：成立、生效、履行、抗辩、解除、违约。
4. 担保看顺位：保证期间、抵押登记、质押交付、留置优先。
5. 公司看外观：章程、决议、登记、代表权限、股东责任。
6. 破产看公平：个别执行让位于集体清偿。
7. 票据看无因：无因性原则 + 直接前后手例外。
8. 保险看代位：赔付后保险人以自己名义追第三者。
9. 民诉看出口：谁告谁、在哪告、谁举证、怎么执行。
10. 主观题看闭环：请求权基础 + 构成要件 + 事实涵摄 + 抗辩 + 结论。
