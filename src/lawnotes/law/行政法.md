# 一、行政法总框架

行政法解决的是：**行政机关如何依法管理社会，公民如何监督和救济行政权。**

法考行政法主要考：

1. 行政法基本原则；
2. 行政主体；
3. 行政行为；
4. 行政许可；
5. 行政处罚；
6. 行政强制；
7. 政府信息公开；
8. 行政复议；
9. 国家赔偿；
10. 行政诉讼衔接。

一句话：  
**行政法看权力来源、程序合法、裁量合理、救济路径。**

# 二、行政法基本原则

## 1. 合法行政原则

核心：**法无授权不可为，法定职责必须为。**

行政机关行使权力必须有法律依据，不能自己创设权力。

常考：

- 行政机关不能超越职权；
- 不能违反上位法；
- 不能不履行法定职责；
- 行政处罚、强制、许可尤其强调法定。

## 2. 合理行政原则

核心是行政裁量不能任性。

包括：

- 公平公正；
- 同等情况同等处理；
- 比例原则；
- 不得明显不当。

比例原则三层：

1. 目的正当；
2. 手段适当；
3. 损害最小；
4. 手段与目的之间利益均衡。

法考常见问法：  
行政机关处罚明显过重、选择最严厉手段、区别对待同类对象，可能违反合理行政。

## 3. 程序正当原则

包括：

- 行政公开；
- 公众参与；
- 听取陈述申辩；
- 回避；
- 说明理由。

程序违法是行政行为被撤销的重要理由。

## 4. 高效便民原则

行政机关应提高效率，便利相对人。

常考：

- 一次性告知；
- 限期办结；
- 简化程序；
- 不得重复提交材料。

## 5. 诚实守信原则

行政机关应守信用。

重点是**信赖保护原则**：

行政机关已经作出的合法有效行政行为，不得随意撤销、变更；确因公共利益需要改变的，应依法补偿相对人损失。

## 6. 权责统一原则

有权必有责，用权受监督，违法要追责。

# 三、行政主体

行政主体是能以自己名义行使行政权，并独立承担法律后果的组织。

## 1. 行政机关

如：

- 国务院；
- 各级人民政府；
- 公安、市场监管、税务、自然资源、生态环境等部门。

## 2. 法律、法规、规章授权组织

被授权组织可以以自己名义作出行政行为，并承担法律后果。

例：

- 法律法规授权的事业单位；
- 行业管理组织；
- 特定公共管理组织。

## 3. 受委托组织

受委托组织不是行政主体。

特点：

- 以委托机关名义实施行政行为；
- 后果由委托机关承担；
- 不得再委托。

法考高频区分：

|类型|是否行政主体|以谁名义|谁担责|行政诉讼被告|
|---|---|---|---|---|
|授权|是|自己|自己|被授权组织|
|委托|否|委托机关|委托机关|委托机关|

# 四、行政行为

## 1. 行政行为概念

行政主体行使行政职权，对外产生法律效果的行为。

法考重点是判断某行为是否可复议、可诉。

## 2. 行政行为分类

按对象：

- 抽象行政行为；
- 具体行政行为。

按是否依申请：

- 依职权行政行为；
- 依申请行政行为。

按效果：

- 授益行政行为，如许可、给付；
- 负担行政行为，如处罚、强制。

按裁量空间：

- 羁束行政行为；
- 裁量行政行为。

## 3. 行政行为效力

包括：

- 公定力；
- 确定力；
- 拘束力；
- 执行力。

重点：行政行为即使有瑕疵，在被撤销、确认违法或无效前，原则上推定有效。

## 4. 行政行为违法情形

常见：

- 主要证据不足；
- 适用法律错误；
- 违反法定程序；
- 超越职权；
- 滥用职权；
- 明显不当；
- 不履行法定职责。

## 5. 行政行为无效

严重且明显违法的行政行为可能无效。

常见：

- 实施主体不具有行政主体资格；
- 没有依据；
- 内容客观上不可能实施；
- 明显重大违法。

# 五、行政许可

行政许可是行政机关根据申请，经审查，准予相对人从事特定活动的行为。

关键词：**依申请、事前控制、授益行为。**

## 1. 可以设定行政许可的事项

常见：

- 涉及国家安全、公共安全、生态环境保护、人身健康、生命财产安全；
- 有限自然资源开发利用、公共资源配置；
- 特定行业市场准入；
- 特定资格、资质；
- 特定设备、设施、产品检验检测检疫；
- 企业或组织设立需要确定主体资格。

## 2. 不应设定行政许可的事项

如果通过以下方式可以解决，就不应设许可：

- 公民、法人能够自主决定；
- 市场竞争机制能够调节；
- 行业组织、中介机构能够自律管理；
- 行政机关采用事后监督能够解决。

## 3. 行政许可设定权限

|文件|能否设定行政许可|
|---|---|
|法律|可以|
|行政法规|可以|
|国务院决定|必要时可以|
|地方性法规|可以设部分许可|
|省级政府规章|可设临时许可|
|部门规章|不得设定行政许可|
|其他规范性文件|不得设定|

省级政府规章设临时许可，满一年需继续实施的，应提请制定地方性法规。

## 4. 行政许可实施程序

基本流程：

申请 → 受理 → 审查 → 决定 → 送达。

重点：

- 材料不齐全，应一次性告知；
- 符合法定条件，应准予许可；
- 不予许可，应说明理由，并告知救济权利；
- 需要听证的，应依法听证。

## 5. 行政许可撤销、撤回、注销

撤销：许可本身取得或作出有违法。  
如行政机关违法作出许可，或申请人欺骗、贿赂取得许可。

撤回：许可合法，但因法律政策变化或公共利益需要收回，应补偿。

注销：许可失效后的登记处理。  
如期限届满未延续、法人终止、许可被撤销吊销等。

# 六、行政处罚

行政处罚是行政机关对违反行政管理秩序的行为给予制裁。

关键词：**违法、制裁、负担行为。**

## 1. 行政处罚种类

包括：

- 警告、通报批评；
- 罚款；
- 没收违法所得、没收非法财物；
- 暂扣许可证件、降低资质等级、吊销许可证件；
- 限制开展生产经营活动；
- 责令停产停业；
- 责令关闭；
- 限制从业；
- 行政拘留；
- 法律、行政法规规定的其他处罚。

## 2. 行政处罚设定权限

|文件|可设定处罚|
|---|---|
|法律|各类行政处罚|
|行政法规|除限制人身自由以外的处罚|
|地方性法规|除限制人身自由、吊销营业执照以外的处罚|
|部门规章、地方政府规章|警告、通报批评、一定数额罚款|

限制人身自由的行政处罚，只能由法律设定。

## 3. 行政处罚适用规则

重点：

- 处罚法定；
- 过罚相当；
- 一事不再罚款；
- 教育与处罚结合；
- 违法行为轻微并及时改正，没有造成危害后果的，不予处罚；
- 初次违法且危害后果轻微并及时改正的，可以不予处罚；
- 当事人有证据足以证明没有主观过错的，不予处罚，法律另有规定除外。

## 4. 处罚追究时效

一般违法行为在 **2 年内**未被发现的，不再处罚。  
涉及公民生命健康安全、金融安全且有危害后果的，期限为 **5 年**。  
法律另有规定除外。

期限从违法行为发生之日起算；连续或继续状态的，从行为终了之日起算。

## 5. 行政处罚程序

### 简易程序

适用于事实确凿、有法定依据、处罚较轻的案件。

可当场作出处罚决定。

### 普通程序

一般处罚案件适用。

流程：

立案 → 调查取证 → 告知 → 陈述申辩 → 法制审核，特定重大案件 → 决定 → 送达 → 执行。

### 听证程序

当事人有权要求听证的典型处罚：

- 较大数额罚款；
- 没收较大数额违法所得、非法财物；
- 降低资质等级；
- 吊销许可证件；
- 责令停产停业；
- 责令关闭；
- 限制从业；
- 其他较重处罚。

听证费用由行政机关承担。

## 6. 行政处罚执行

罚缴分离原则：作出处罚决定的机关与收缴罚款的机构分离。

例外情况下可当场收缴。

当事人逾期不履行，行政机关可：

- 加处罚款；
- 申请法院强制执行；
- 依法采取其他措施。

# 七、行政强制

行政强制分为：

1. 行政强制措施；
2. 行政强制执行。

## 1. 行政强制措施

行政机关为制止违法、防止证据损毁、避免危害、控制危险，对人身或财物进行暂时控制。

种类：

- 限制公民人身自由；
- 查封场所、设施或财物；
- 扣押财物；
- 冻结存款、汇款；
- 其他行政强制措施。

特点：临时性、预防性、控制性。

## 2. 行政强制执行

行政机关或法院对不履行行政决定的相对人，依法强制其履行义务。

方式：

- 加处罚款或滞纳金；
- 划拨存款、汇款；
- 拍卖、依法处理查封扣押财物；
- 排除妨碍、恢复原状；
- 代履行；
- 其他方式。

## 3. 设定权限

行政强制措施原则上由法律设定。

行政强制执行只能由法律设定。

行政机关没有强制执行权的，应申请人民法院强制执行。

## 4. 行政强制措施程序

一般要求：

- 由两名以上行政执法人员实施；
- 出示执法证件；
- 通知当事人到场；
- 告知理由、依据和权利；
- 听取陈述申辩；
- 制作现场笔录；
- 当场交付决定书和清单。

紧急情况可先实施，事后补办批准手续。

## 5. 查封、扣押

对象限于涉案场所、设施或财物。

不得查封、扣押与违法行为无关的财物。  
不得查封、扣押公民个人及其扶养家属的生活必需品。

期限一般不得超过 **30 日**；情况复杂可延长，但延长期限一般也不得超过 **30 日**，法律、行政法规另有规定除外。

## 6. 冻结

冻结存款、汇款应由法律规定的行政机关实施。

不得重复冻结。

## 7. 代履行

适用于行政相对人不履行排除妨碍、恢复原状等义务，后果已经或将危害交通安全、环境污染、自然资源破坏等。

代履行费用由义务人承担。

# 八、政府信息公开

政府信息公开属于行政法高频小考点。

## 1. 基本原则

以公开为常态，不公开为例外。

## 2. 公开方式

包括：

- 主动公开；
- 依申请公开。

## 3. 不予公开

不得公开：

- 国家秘密；
- 法律、行政法规禁止公开的信息；
- 公开后可能危及国家安全、公共安全、经济安全、社会稳定的信息。

涉及商业秘密、个人隐私等，公开会损害第三方合法权益的，一般不公开；但第三方同意公开，或行政机关认为不公开会对公共利益造成重大影响的，可以公开。

## 4. 依申请公开处理

行政机关收到申请后，应依法答复：

- 属于公开范围的，提供信息；
- 不属于本机关负责的，告知并说明；
- 信息不存在的，告知不存在；
- 属于不予公开的，说明理由；
- 申请内容不明确的，告知补正。

# 九、行政复议

行政复议是近年行政法最重要新增重点之一。新法强调：**发挥行政复议化解行政争议的主渠道作用。**

## 1. 行政复议性质

行政复议是行政机关内部纠错和救济制度。

特点：

- 审查行政行为合法性；
- 也审查适当性；
- 程序相对便捷；
- 可调解；
- 与行政诉讼衔接。

## 2. 可申请复议的事项

常见：

- 行政处罚；
- 行政强制；
- 行政许可；
- 行政确认；
- 行政征收、征用及补偿；
- 行政给付；
- 行政不作为；
- 政府信息公开；
- 行政协议；
- 侵犯经营自主权、农村土地承包经营权等。

## 3. 不属于复议范围

典型：

- 国防、外交等国家行为；
- 行政法规、规章等抽象规范本身；
- 行政机关内部人事处理；
- 行政机关对民事纠纷作出的调解等，法律另有规定除外。

## 4. 复议机关

一般规则：

- 对县级以上地方政府工作部门行为不服，通常向本级人民政府申请；
- 对地方各级人民政府行为不服，向上一级人民政府申请；
- 对国务院部门或省级政府行为不服，向作出行为的机关申请复议；对复议决定不服，可诉讼，也可申请国务院裁决，国务院裁决为最终裁决。

新行政复议法调整较多，法考重点是具体题干中识别复议机关。

## 5. 复议申请期限

一般：知道或应当知道行政行为之日起 **60 日内**提出。  
法律另有规定除外。

## 6. 复议审理

复议原则：

- 合法；
- 公正；
- 公开；
- 高效；
- 便民；
- 为民。

行政复议可以调解。

## 7. 复议决定类型

包括：

- 维持；
- 撤销；
- 变更；
- 确认违法；
- 确认无效；
- 责令履行；
- 驳回复议申请；
- 驳回复议请求等。

## 8. 复议前置

一般情况下，行政相对人可以选择：

- 直接行政诉讼；
- 先复议再诉讼。

但法律规定复议前置的，必须先复议。

常见复议前置：

- 自然资源所有权、使用权确权争议；
- 纳税争议；
- 法律规定的其他事项。

# 十、国家赔偿

国家赔偿分为：

1. 行政赔偿；
2. 刑事赔偿；
3. 非刑事司法赔偿。

行政法中重点是行政赔偿。

## 1. 行政赔偿构成

要件：

- 行政机关及其工作人员行使职权；
- 行为违法；
- 造成合法权益损害；
- 行为与损害之间有因果关系。

## 2. 行政赔偿范围

侵犯人身权：

- 违法拘留；
- 违法限制人身自由；
- 非法拘禁或以其他方法非法剥夺人身自由；
- 殴打、虐待等造成伤害或死亡；
- 违法使用武器、警械造成伤害或死亡。

侵犯财产权：

- 违法罚款；
- 违法没收财物；
- 违法吊销许可证和执照；
- 违法责令停产停业；
- 违法查封、扣押、冻结；
- 违法征收、征用财产等。

## 3. 不赔情形

常见：

- 行政机关工作人员与职权无关的个人行为；
- 因公民、法人、其他组织自己的行为致损害；
- 法律规定的其他情形。

## 4. 赔偿义务机关

一般：作出侵权行政行为的机关。

两个以上机关共同侵权的，共同赔偿。

经复议的：

- 复议机关维持原行为，原机关赔偿；
- 复议机关加重损害的，复议机关对加重部分赔偿。

## 5. 赔偿方式

以支付赔偿金为主。

也可：

- 返还财产；
- 恢复原状；
- 消除影响；
- 恢复名誉；
- 赔礼道歉。

# 十一、行政法与行政诉讼衔接

行政法主观题常问：行政行为是否合法？相对人怎么救济？

答题顺序：

1. **看主体**  
    作出行为的是行政主体吗？是授权还是委托？
    
2. **看权限**  
    是否有法定职权？有没有超越职权？
    
3. **看事实证据**  
    主要事实是否清楚？证据是否充分？
    
4. **看法律依据**  
    适用法律是否正确？
    
5. **看程序**  
    是否告知、听证、说明理由、送达？
    
6. **看裁量**  
    是否明显不当？是否违反比例原则？
    
7. **看救济**  
    复议、诉讼、赔偿如何选择？

# 十二、法考高频比较表

## 1. 行政许可、行政处罚、行政强制

|项目|行政许可|行政处罚|行政强制|
|---|---|---|---|
|性质|授益行为|制裁行为|控制或执行行为|
|启动|依申请|多依职权|多依职权|
|目的|准入、资格确认|惩戒违法|制止、防控、执行|
|典型|许可经营|罚款、吊销|查封、扣押、代履行|

## 2. 撤销、撤回、注销

|类型|原因|是否违法|后果|
|---|---|---|---|
|撤销|行政行为作出时有违法|是|可溯及既往|
|撤回|行政行为合法，但公共利益需要改变|否|通常应补偿|
|注销|行政行为失效后的登记处理|不一定|消灭存续状态|

## 3. 行政强制措施 vs 行政强制执行

|项目|强制措施|强制执行|
|---|---|---|
|时间|行政决定前或过程中|行政决定后|
|目的|控制危险、保全证据|迫使履行义务|
|特点|临时性|执行性|
|例子|查封、扣押、冻结|划拨、拍卖、代履行|

## 4. 授权 vs 委托

|项目|授权|委托|
|---|---|---|
|来源|法律、法规、规章|行政机关委托|
|主体资格|有行政主体资格|无行政主体资格|
|名义|自己名义|委托机关名义|
|后果|自己承担|委托机关承担|

# 十三、复习优先级

如果时间有限，行政法按这个顺序复习：

1. 行政法基本原则；
2. 行政主体：授权与委托；
3. 行政行为效力、违法、无效；
4. 行政处罚：种类、设定、程序、听证、一事不再罚；
5. 行政许可：设定、实施、撤销撤回注销；
6. 行政强制：措施与执行、程序、查封扣押冻结；
7. 行政复议：范围、机关、期限、决定；
8. 政府信息公开；
9. 国家赔偿；
10. 行政诉讼衔接。

行政法最核心一句话：  
**先问有没有权，再问程序对不对，再问裁量是否合理，最后问怎么救济。**
