下面单独整理**理论法**。按法考口径，理论法通常包括：

1. **习近平法治思想**
2. **法理学**
3. **宪法**
4. **中国法律史**
5. **司法制度和法律职业道德**

2026 年法考公告仍将这些内容放在客观题试卷一中，且继续重点考查习近平法治思想。参考：[司法部 2026 年法考公告](https://www.moj.gov.cn/pub/sfbgw/zwxxgk/fdzdgknr/fdzdgknrtzwj/202606/t20260605_536058.html)、[2026 年法考大纲发布说明](https://app.xinhuanet.com/news/article.html?articleId=202605154aec04d86ee24219a8c097d0c6b9cc46)。

# 一、理论法总框架

理论法的特点：

- **习近平法治思想**：政治方向题，重在准确记忆。
- **法理学**：概念、分类、法律推理、法律解释。
- **宪法**：国家制度、公民权利、国家机构。
- **中国法律史**：朝代法制、代表法典、重要制度。
- **司法制度与法职**：法官、检察官、律师、公证、法律援助、仲裁、司法责任。

理论法复习策略：

1. 先背框架；
2. 再记关键词；
3. 最后刷易混点。

# 第一编 习近平法治思想

# 二、习近平法治思想的地位

习近平法治思想是：

- 马克思主义法治理论中国化时代化的重大成果；
- 习近平新时代中国特色社会主义思想的重要组成部分；
- 全面依法治国的根本遵循和行动指南。

法考关键词：

- 党的领导；
- 人民主体；
- 中国特色社会主义法治道路；
- 依宪治国、依宪执政；
- 法治体系；
- 法治国家、法治政府、法治社会一体建设；
- 科学立法、严格执法、公正司法、全民守法；
- 涉外法治；
- 法治人才队伍。

1. 坚持党对全面依法治国的领导；
2. 坚持以人民为中心；
3. 坚持中国特色社会主义法治道路；
4. 坚持依宪治国、依宪执政；
5. 坚持在法治轨道上推进国家治理体系和治理能力现代化；
6. 坚持建设中国特色社会主义法治体系；
7. 坚持依法治国、依法执政、依法行政共同推进；
8. 坚持法治国家、法治政府、法治社会一体建设；
9. 坚持全面推进科学立法、严格执法、公正司法、全民守法；
10. 坚持统筹推进国内法治和涉外法治；
11. 坚持建设德才兼备的高素质法治工作队伍。

高频对应：

- **根本保证**：党的领导。

- **唯一正确道路**：中国特色社会主义法治道路。

- **最后一道防线**：公正司法。

# 五、科学立法、严格执法、公正司法、全民守法

## 1. 科学立法

要求：

- 科学立法；
- 民主立法；
- 依法立法。

重点：

- 立法要体现人民意志；
- 立法要符合宪法；
- 立法要回应实践需要；
- 立法质量优先于数量。

## 2. 严格执法

要求：

- 规范；
- 公正；
- 文明。

重点：

- 行政机关必须依法行政；
- 法无授权不可为；
- 法定职责必须为；
- 执法不得选择性、运动式、逐利式。

## 3. 公正司法

公正司法是维护社会公平正义的最后一道防线。

要求：

- 司法公正；
- 司法公开；
- 司法责任制；
- 保障人权；
- 努力让人民群众在每一个司法案件中感受到公平正义。

## 4. 全民守法

全民守法是法治社会基础。

重点：

- 领导干部带头尊法学法守法用法；
- 增强全民法治观念；
- 建设社会主义法治文化；
- 把法治教育纳入国民教育体系。

# 第二编 法理学

# 六、法的概念与特征

法是由国家制定或认可，并由国家强制力保证实施的行为规范体系。

基本特征：

1. 规范性；
2. 国家意志性；
3. 国家强制性；
4. 普遍性；
5. 程序性；
6. 可诉性。

法与道德区别：

|项目|法|道德|
|---|---|---|
|产生方式|国家制定或认可|社会自发形成|
|表现形式|成文或认可规则|观念、习俗、舆论|
|实施保障|国家强制力|内心信念、社会评价|
|调整范围|相对有限|更广泛|
|程序性|强|弱|

# 七、法的作用

## 1. 规范作用

包括：

- 指引作用；
- 评价作用；
- 预测作用；
- 教育作用；
- 强制作用。

区分：

- 指引：指导本人行为；
- 评价：评价他人行为是否合法；
- 预测：预判他人或机关如何反应；
- 教育：通过法律实施影响社会公众；
- 强制：制裁违法犯罪。

## 2. 社会作用

包括：

- 维护社会秩序；
- 保障权利；
- 分配利益；
- 解决纠纷；
- 推进治理。

---

# 八、法的价值

法的主要价值：

- 秩序；
- 自由；
- 平等；
- 公平；
- 正义；
- 效率；
- 人权。

法考常考价值冲突。

处理方式：

1. 价值位阶；
2. 个案平衡；
3. 比例原则；
4. 利益衡量。

注意：

- 秩序是基础价值；
- 自由是重要价值；
- 正义是核心价值；
- 人权是现代法治的重要价值目标。

# 九、法律规则与法律原则

## 1. 法律规则

法律规则结构：

- 假定条件；
- 行为模式；
- 法律后果。

分类：

按内容：

- 授权性规则；
- 义务性规则；
- 禁止性规则。

按强制程度：

- 强行性规则；
- 任意性规则。

按确定程度：

- 确定性规则；
- 委任性规则；
- 准用性规则。

## 2. 法律原则

法律原则特点：

- 抽象性；
- 概括性；
- 稳定性；
- 适用时需要衡量。

法律原则与规则区别：

|项目|法律规则|法律原则|
|---|---|---|
|内容|具体明确|抽象概括|
|适用方式|全有或全无|衡量适用|
|覆盖范围|较窄|较广|
|冲突处理|特别法、上位法等|价值衡量|

# 十、法的渊源与效力

## 1. 我国正式法源

包括：

- 宪法；
- 法律；
- 行政法规；
- 监察法规；
- 地方性法规；
- 自治条例、单行条例；
- 规章；
- 特别行政区法律；
- 国际条约、国际惯例。

## 2. 效力位阶

基本规则：

- 宪法具有最高法律效力；
- 法律高于行政法规；
- 行政法规高于地方性法规、规章；
- 地方性法规高于本级和下级地方政府规章；
- 部门规章与地方政府规章效力位阶一般相同。

冲突处理：

- 上位法优于下位法；
- 特别法优于一般法；
- 新法优于旧法；
- 同一机关制定的新的一般规定与旧的特别规定不一致，由制定机关裁决。

---

# 十一、法律关系

法律关系三要素：

1. 主体；
2. 内容；
3. 客体。

法律关系主体：

- 自然人；
- 法人；
- 非法人组织；
- 国家等。

法律关系内容：

- 权利；
- 义务；
- 职权；
- 职责。

法律关系客体：

- 物；
- 行为；
- 智力成果；
- 人身利益；
- 信息等。

法律事实：

- 法律事件：不以人的意志为转移，如出生、死亡、自然灾害；
- 法律行为：以人的意志为转移，如签合同、侵权、犯罪。

---

# 十二、法律责任

法律责任类型：

- 民事责任；
- 行政责任；
- 刑事责任；
- 宪法责任；
- 国家赔偿责任。

法律责任归责原则：

1. 责任法定；
2. 因果联系；
3. 责任相称；
4. 责任自负；
5. 公正归责。

免责事由：

- 时效免责；
- 不诉免责；
- 自首、立功等减轻或免除；
- 正当防卫、紧急避险；
- 不可抗力；
- 有效补救等。

---

# 十三、法律解释

法律解释按主体：

- 立法解释；
- 司法解释；
- 行政解释；
- 学理解释。

按方法：

- 文义解释；
- 体系解释；
- 目的解释；
- 历史解释；
- 扩张解释；
- 限缩解释；
- 当然解释；
- 反对解释。

高频点：

- 文义解释是起点；
- 目的解释常用于疑难案件；
- 体系解释强调条文之间协调；
- 刑法中禁止类推入罪；
- 有利于被告人的解释更容易被接受。

---

# 十四、法律推理

类型：

1. 演绎推理；
2. 归纳推理；
3. 类比推理；
4. 设证推理；
5. 辩证推理。

法考重点：

- 三段论是演绎推理；
- 从个案总结一般规则是归纳；
- 类似案件类似处理是类比；
- 疑难案件中多价值冲突，需要辩证推理。

# 第三编 宪法

# 十六、宪法基本理论

宪法是国家根本法。

宪法特征：

- 内容根本性；
- 效力最高性；
- 制定修改程序严格；
- 监督实施特殊性。

我国现行宪法是 **1982 年宪法**。

宪法基本原则：

- 党的领导原则；
- 人民主权原则；
- 基本人权原则；
- 法治原则；
- 民主集中制原则。

---

# 十七、国家性质和基本制度

## 1. 国体

我国国体：

**人民民主专政的社会主义国家。**

## 2. 政体

我国政体：

**人民代表大会制度。**

## 3. 根本制度与根本政治制度

根本制度：

- 社会主义制度。

根本政治制度：

- 人民代表大会制度。

基本政治制度：

- 中国共产党领导的多党合作和政治协商制度；
- 民族区域自治制度；
- 基层群众自治制度。

## 4. 基本经济制度

包括：

- 公有制为主体、多种所有制经济共同发展；
- 按劳分配为主体、多种分配方式并存；
- 社会主义市场经济体制。

---

# 十八、选举制度

基本原则：

- 普遍性原则；
- 平等性原则；
- 直接选举与间接选举并用；
- 秘密投票；
- 差额选举。

直接选举：

- 县级人大代表；
- 乡级人大代表。

间接选举：

- 全国人大代表；
- 省级人大代表；
- 设区的市、自治州人大代表。

选民资格：

- 中国公民；
- 年满 18 周岁；
- 未被依法剥夺政治权利。

# 十九、国家结构形式

我国是单一制国家。

## 1. 民族区域自治

自治地方：

- 自治区；
- 自治州；
- 自治县。

不包括民族乡。

自治机关：

- 自治地方的人民代表大会；
- 自治地方的人民政府。

不包括法院、检察院。

## 2. 特别行政区

特别行政区实行高度自治，但中央享有全面管治权。

特别行政区享有：

- 行政管理权；
- 立法权；
- 独立的司法权和终审权。

不享有：

- 外交权；
- 国防权。

---

# 二十、公民基本权利和义务

## 1. 平等权

公民在法律面前一律平等。

注意：平等不是绝对相同，而是同等情况同等对待，不同情况合理区别。

## 2. 政治权利和自由

包括：

- 选举权和被选举权；
- 言论、出版、集会、结社、游行、示威自由。

## 3. 宗教信仰自由

国家保护正常宗教活动。

任何人不得利用宗教破坏社会秩序、损害公民身体健康、妨碍国家教育制度。

## 4. 人身自由

包括：

- 人身自由不受侵犯；
- 人格尊严不受侵犯；
- 住宅不受侵犯；
- 通信自由和通信秘密受保护。

## 5. 社会经济权利

包括：

- 财产权；
- 劳动权；
- 休息权；
- 退休人员生活保障权；
- 物质帮助权。

劳动权既是权利，也是义务。

## 6. 文化教育权利

包括：

- 受教育权；
- 科学研究、文学艺术创作和其他文化活动自由。

受教育既是权利，也是义务。

## 7. 监督权

包括：

- 批评权；
- 建议权；
- 申诉权；
- 控告权；
- 检举权；
- 取得国家赔偿权。

---

# 二十一、国家机构

## 1. 全国人大

性质：

- 最高国家权力机关。

职权：

- 修改宪法；
- 监督宪法实施；
- 制定和修改基本法律；
- 选举国家主席、副主席；
- 选举中央军委主席、国家监察委主任、最高法院长、最高检检察长；
- 决定国务院总理人选；
- 批准国民经济和社会发展计划、国家预算等。

## 2. 全国人大常委会

职权：

- 解释宪法；
- 监督宪法实施；
- 制定和修改非基本法律；
- 解释法律；
- 决定特赦；
- 决定战争状态；
- 决定全国总动员或局部动员；
- 监督国务院、监察委、最高法、最高检工作。

## 3. 国家主席

国家主席根据全国人大及其常委会决定，公布法律、任免国务院组成人员、授予国家勋章和荣誉称号、发布特赦令等。

国家主席本身不独立决定重大国家事务。

## 4. 国务院

最高国家行政机关。

职权：

- 制定行政法规；
- 领导各部委和地方政府；
- 编制执行国民经济和社会发展计划、国家预算；
- 管理经济、教育、科学、文化、卫生、公安、民政等行政事务。

## 5. 国家监察委员会

最高监察机关。

监察对象：

- 所有行使公权力的公职人员。

对全国人大及其常委会负责，并接受监督。

## 6. 人民法院和人民检察院

人民法院：国家审判机关。  
人民检察院：国家法律监督机关。

最高人民法院对全国人大及其常委会负责。  
最高人民检察院对全国人大及其常委会负责。

---

# 二十二、宪法监督

全国人大及其常委会监督宪法实施。

全国人大常委会有权解释宪法。

备案审查是宪法监督的重要方式。

重点：

- 行政法规、地方性法规、自治条例、单行条例、规章等应依法备案；
- 与宪法、法律相抵触的，应予撤销或纠正。

# 第四编 中国法律史

# 二十三、中国法律史复习方法

中国法律史分值不大，但容易拿分。按朝代记：

**西周礼刑 → 春秋战国成文法 → 秦汉律令 → 魏晋南北朝儒家化 → 隋唐成熟 → 宋元明清发展 → 清末变法 → 民国六法。**

# 二十四、西周法制

关键词：

- 明德慎罚；
- 礼刑结合；
- 宗法等级；
- 亲亲、尊尊；
- 五听断狱。

五听：

- 辞听；
- 色听；
- 气听；
- 耳听；
- 目听。

“礼不下庶人，刑不上大夫”表示礼刑适用有等级差异，不是贵族完全不受刑罚。

# 二十五、春秋战国法制

重要事件：

- 郑国子产铸刑书；

# 第五编 司法制度和法律职业道德

# 三十二、司法制度总论

中国特色社会主义司法制度包括：

- 审判制度；
- 检察制度；
- 律师制度；
- 公证制度；
- 法律援助制度；
- 人民调解制度；
- 仲裁制度。

司法基本要求：

- 司法公正；
- 司法效率；
- 司法公开；
- 司法责任制；
- 人权保障；
- 接受监督。

---

# 三十三、审判制度

人民法院是国家审判机关。

审判原则：

- 依法独立行使审判权；
- 以事实为根据，以法律为准绳；
- 公开审判为原则，不公开为例外；
- 两审终审；
- 人民陪审员制度；
- 审判责任制。

法院体系：

- 最高人民法院；
- 地方各级人民法院；
- 专门人民法院。

---

# 三十四、检察制度

人民检察院是国家法律监督机关。

主要职能：

- 刑事检察；
- 民事检察；
- 行政检察；
- 公益诉讼检察；
- 法律监督。

检察机关职权：

- 批准、决定逮捕；
- 提起公诉；
- 侦查监督；
- 审判监督；
- 执行监督；
- 公益诉讼。

---

# 三十五、法官职业道德

核心要求：

- 忠诚司法事业；
- 保证司法公正；
- 确保司法廉洁；
- 坚持司法为民；
- 维护司法形象。

法官不得：

- 私下会见当事人、律师、特殊关系人；
- 接受请托、送礼、宴请；
- 泄露审判秘密；
- 违规过问案件；
- 插手他人案件；
- 从事营利活动；
- 兼任律师、仲裁员等影响公正的职务。

---

# 三十六、检察官职业道德

核心要求：

- 忠诚；
- 为民；
- 担当；
- 公正；
- 廉洁。

检察官不得：

- 徇私枉法；
- 刑讯逼供或非法取证；
- 泄露国家秘密、检察工作秘密；
- 接受当事人请托；
- 违规干预案件；
- 从事营利性活动。

---

# 三十七、律师制度与职业道德

律师职责：

- 维护当事人合法权益；
- 维护法律正确实施；
- 维护社会公平正义。

律师权利：

- 会见；
- 阅卷；
- 调查取证；
- 发问、质证、辩论；
- 代理申诉、控告；
- 依法获得执业保障。

律师义务：

- 勤勉尽责；
- 保守秘密；
- 避免利益冲突；
- 不得虚假承诺；
- 不得伪造证据；
- 不得扰乱法庭；
- 不得私自收费；
- 不得行贿。

## 利益冲突

律师不得在同一案件中为双方当事人担任代理人或辩护人。

同一律师事务所也应避免利益冲突，法律另有规定或当事人同意仍需符合法定要求。

## 保密义务

律师对执业中知悉的国家秘密、商业秘密、个人隐私和当事人不愿公开情况负保密义务。

但当事人或他人准备、正在实施危害国家安全、公共安全以及严重危害他人人身安全犯罪的，律师不得保密。

---

# 三十八、公证制度

公证机构依法证明民事法律行为、有法律意义的事实和文书的真实性、合法性。

公证效力：

- 证据效力；
- 强制执行效力，限依法赋予强制执行效力的债权文书；
- 法律行为成立或生效要件效力，限法律规定或当事人约定。

公证员不得：

- 为本人及近亲属办理公证；
- 私自出证；
- 违规收费；
- 泄露秘密；
- 从事有利益冲突的活动。

---

# 三十九、法律援助制度

法律援助对象：

- 经济困难公民；
- 符合法定情形的特殊案件当事人。

刑事案件应通知法律援助的常见情形：

- 未成年人；
- 盲、聋、哑人；
- 尚未完全丧失辨认或控制能力的精神病人；
- 可能被判处无期徒刑、死刑；
- 缺席审判案件被告人；
- 法律规定的其他情形。

法律援助机构、律师不得向受援人收取财物。

---

# 四十、人民调解、仲裁与基层法律服务

## 1. 人民调解

人民调解委员会调解民间纠纷。

特点：

- 自愿；
- 平等；
- 不收费；
- 不得违反法律法规和国家政策。

调解协议具有民事合同性质。  
经司法确认后，具有强制执行效力。

## 2. 仲裁

特点：

- 自愿；
- 专业；
- 独立；
- 一裁终局；
- 不公开为原则。

仲裁协议有效要件：

- 有请求仲裁的意思表示；
- 有仲裁事项；
- 有选定的仲裁委员会。

婚姻、收养、监护、扶养、继承纠纷不得仲裁。

# 四十二、理论法复习优先级

如果时间有限，按这个顺序背：

1. 习近平法治思想“十一个坚持”；
2. 全面依法治国总目标和法治体系；
3. 宪法国家制度、公民权利、国家机构；
4. 法理学：规则原则、法源效力、解释推理、法律关系；
5. 法律职业道德：法官、检察官、律师禁止性规则；
6. 中国法律史：法典、朝代、制度关键词。

理论法拿分核心：

**政治表述要准，宪法制度要细，法理概念要会区分，法史抓关键词，法职抓禁止性规则。**
